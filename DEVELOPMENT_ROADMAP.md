### 任务清单

| # | 任务 | 状态 | 负责人 |
|---|------|------|--------|
| P0.1 | Supabase PostgreSQL 数据库配置 | 🔄 | Chief Engineer |
| P0.2 | SQLAlchemy 2.0 ORM + asyncpg 异步引擎 | ⬜ | Chief Engineer |
| P0.3 | Alembic 迁移工具配置 + 核心模型迁移 | ⬜ | Chief Engineer |
| P0.4 | JWT access/refresh token 双令牌认证架构 | ⬜ | Chief Engineer |

| P0.7 | 单元测试 + 集成测试 + P0 验收报告 | ⬜ | Chief Engineer |
